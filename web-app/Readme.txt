ЗАПИСНАЯ КНИЖКА

Веб-приложение для хранения контактных данных пользователей: имени, фамилии, возраста, пола и номера телефона. Приложение позволяет добавлять пользователей, редактировать их данные, а также производить контекстный поиск по одному из параметров.


ТРЕБОВАНИЯ
Для установки и использования приложения необходимо установить JDK не ниже 8.0 версии, Apache Maven не ниже 3.0,
база данных MariaDB не ниже 10 версии с пользователем под именем app и паролем qwerty, которая должна работать по адресу localhost:3306.

СБОРКА
Разархивируйте проект в вашу папку, откройте командную строку, перейдите в корневой каталог проекта и введите:

mvn clean package

После этого файл собранного приложения web-app.war можно найти в папке target корневого каталога. Автоматически создастся база данных "user_db", в ней таблица "users", которая заполнится тестовыми данными.


ЗАПУСК
Чтобы запустить проект, в командной строке перейдите в корневой каталог проекта и введите:

mvn jetty:run

При запуске также генерируется база данных (если ее еще нет), заполняется тестовыми данными, а также производится
сборка проекта если он еще не собран. После этого приложение доступно в браузере по ссылке:

http://localhost:8080


АВТОМАТИЧЕСКАЯ СБОРКА И ЗАПУСК(только для Windows)
Чтобы собрать приложение, произведите двойной клик мышью по файлу pack.bat, который находится в корне проекта.
Чтобы запустить - по run.bat и приложение будет доступно в браузере по ссылке:

http://localhost:8080/

ОСТАНОВКА
Для остановки приложения перейдите в консоль и нажмите комбинацию клавиш Ctrl+C, а потом Y.
